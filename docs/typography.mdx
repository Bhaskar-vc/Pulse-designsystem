## Font Families

| Variable | Font | Use |
|----------|------|-----|
| `--font-sans` | Inter | All body text, UI labels, headings h1–h5 |
| `--font-display` | Poppins | Display headings (h6), brand text |
| `--font-mono` | JetBrains Mono | Code, tokens, technical labels |

## Weights

| Token | Value | Usage |
|-------|-------|-------|
| `--fw-regular` | 400 | Body, captions |
| `--fw-medium` | 500 | Headings, labels |
| `--fw-semibold` | 600 | Subheadings, table headers |
| `--fw-bold` | 700 | Emphasis, bold body |
