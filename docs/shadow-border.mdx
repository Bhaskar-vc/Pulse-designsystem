## CSS Custom Properties

**Shadows** (Tailwind `boxShadow` + CSS `--ds-shadow-*`)

| Token | Value |
|-------|-------|
| `shadow-xs` / `--ds-shadow-sm` | `0 1px 2px rgba(16,24,40,0.05)` |
| `shadow-sm` | `0 1px 3px rgba(16,24,40,0.10), 0 1px 2px rgba(16,24,40,0.06)` |
| `shadow-md` / `--ds-shadow-md` | `0 4px 8px -2px rgba(16,24,40,0.10), 0 2px 4px -2px rgba(16,24,40,0.06)` |
| `shadow-lg` / `--ds-shadow-lg` | `0 12px 16px -4px rgba(16,24,40,0.08), 0 4px 6px -2px rgba(16,24,40,0.03)` |
| `shadow-xl` | `0 20px 24px -4px rgba(16,24,40,0.08), 0 8px 8px -4px rgba(16,24,40,0.03)` |
| `shadow-2xl` | `0 24px 48px -12px rgba(16,24,40,0.18)` |

**Border Radius** (Tailwind `borderRadius`)

| Token | Value |
|-------|-------|
| `rounded-none` | `0` |
| `rounded-2xs` | `2px` |
| `rounded-xs` | `4px` |
| `rounded-sm` | `6px` |
| `rounded` (DEFAULT) | `8px` |
| `rounded-md` | `10px` |
| `rounded-lg` | `12px` |
| `rounded-xl` | `16px` |
| `rounded-pill` | `50px` |

**Border Width** (Tailwind `borderWidth`)

| Token | Value |
|-------|-------|
| `border-none` | `0` |
| `border` (DEFAULT) | `1px` |
| `border-md` | `1.5px` |
| `border-lg` | `2px` |
