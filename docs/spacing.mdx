## Layout Principles

<div className="sp-principles">
  <div className="sp-principle-card">
    <div className="sp-principle-num">01</div>
    <div className="sp-principle-title">Group by similarity</div>
    <div className="sp-principle-desc">Use consistent spacing between elements that belong to the same group. Uniform gaps signal that items are related, creating visual cohesion without needing borders or labels.</div>
  </div>
  <div className="sp-principle-card">
    <div className="sp-principle-num">02</div>
    <div className="sp-principle-title">Group by proximity</div>
    <div className="sp-principle-desc">The distance between elements creates semantic meaning. Elements that are closer together are perceived as more related — use tighter spacing within components and larger gaps between them.</div>
  </div>
  <div className="sp-principle-card">
    <div className="sp-principle-num">03</div>
    <div className="sp-principle-title">Create hierarchy</div>
    <div className="sp-principle-desc">Use scale and whitespace to rank importance. More whitespace around an element elevates its visual weight — headlines, CTAs, and hero sections deserve larger breathing room.</div>
  </div>
  <div className="sp-principle-card">
    <div className="sp-principle-num">04</div>
    <div className="sp-principle-title">Introduce rhythm</div>
    <div className="sp-principle-desc">Repeating spacing patterns across a layout guide the user's eye. Alternating dense and open areas create a natural reading cadence, making content feel intentional rather than crowded.</div>
  </div>
  <div className="sp-principle-card">
    <div className="sp-principle-num">05</div>
    <div className="sp-principle-title">Use optical adjustment</div>
    <div className="sp-principle-desc">Minor deviations from the standard scale are sometimes necessary to maintain visual harmony — for example, icons often need 1–2 px nudges to appear optically centred despite being mathematically centred.</div>
  </div>
</div>
