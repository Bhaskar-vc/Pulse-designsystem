# Design Tokens

Core visual constants used across all Pulse DS components.

## Colors

### Primary (Brand)
| Token | Hex | Usage |
|-------|-----|-------|
| primary-700 | `#29294c` | Text, solid buttons, headings |
| primary-500 | `#707087` | Secondary text |
| primary-400 | `#9d9dad` | Tertiary text, placeholders |
| primary-100 | `#f4f4f6` | Subtle backgrounds |
| primary-50 | `#f9fafb` | Page background |

### Secondary (Accent)
| Token | Hex | Usage |
|-------|-----|-------|
| secondary-700 | `#6941c6` | Links, active states |
| secondary-600 | `#7f56d9` | Primary accent, focus rings |
| secondary-100 | `#f4ebff` | Accent backgrounds |
| secondary-50 | `#f9f5ff` | Hover backgrounds |

### Semantic
| Token | Hex | Usage |
|-------|-----|-------|
| success | `#039855` | Success states, online indicators |
| warning | `#f79009` | Warning alerts, caution |
| error | `#f04438` | Error states, destructive actions |
| info | `#0ba5ec` | Informational, links |
